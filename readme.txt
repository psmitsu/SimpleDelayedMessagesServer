Тестовый сервер для работы с очередями. Можно добавлять записи со отложенным временем в секундах и удалять их по ключу.
Сервер нужно запускать с аргументом, в котором будет указано имя очереди в Redis, которую он будет обслуживать.
Например node index.js testQueue
По / есть инструкция, как добавлять сообщения и удалять их.
Нужно иметь работающий сервер Redis на локальной машине.
При запуске сервера он создает очередь testQueue в редисе, соответственно, перед следующим запуском ее нужно будет удалить с помощью скрипта deleteQueue.js.
node deleteQueue.js testQueue.
Чтобы слушать очередь, нужно запустить скрипт rsmqListener.js с аргументом название очереди.
node rsmqListener.js testQueue
