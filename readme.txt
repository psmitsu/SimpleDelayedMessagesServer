Тестовый сервер для работы с очередями. Можно добавлять записи со отложенным временем в секундах и удалять их по ключу.
По / есть инструкция, как добавлять сообщения и удалять их.
Нужно иметь работающий сервер Redis на локальной машине.
При запуске сервера он создает очередь testQueue в редисе, соответственно, перед следующим запуском ее нужно будет удалить с помощью скрипта handleRSMQ.js
